# Собственные хосты

Это репозиторий для хранения и передачи `docker-compose.yml`, чтобы каждый желающий мог развернуть собственный `Self-Host` у себя на ПК и сервере

# Планируемая схема
```
    Интернет -> Роутер -> NPM -> Guacamole
                (Порты)    |  |
                           |  └-> Proxmox
                           |
                         Keycloack
```

# *Немного о важном!*
После выполнения команды `git clone https://github.com/H1S0K4-M0R0W/self-host.git` необходимо заполнить поля в файлах `env.example` и удалить окончание `*.example` -> `.env`
